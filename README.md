# Gravichem - Premium Coming Soon Landing Page

A luxury-themed coming soon landing page for Gravichem, featuring premium graphite business aesthetics.

## Features

- **Premium Design**: Black to white gradient with luxury feel
- **Typography**: Times New Roman throughout
- **Animations**: Smooth fade-in effects using Anime.js
- **3D Background**: Subtle graphite particle animation using Three.js
- **Responsive**: Mobile-friendly design

## Tech Stack

- Next.js 14
- Three.js for 3D animations
- Anime.js for smooth transitions
- TypeScript

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
